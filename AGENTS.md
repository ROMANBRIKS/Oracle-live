# Project Instructions

## UI Focus
- The primary focus for upcoming tasks is **redesigning and modifying the UI**.
- Avoid modifying core logic unless explicitly requested or strictly necessary for a UI change.
- Acknowledge that new logic might be added later, but for now, prioritize visual design.

## Maintenance & Stability
- If the application fails to start or shows a "Please wait" loop, refer to `/DB_TROUBLESHOOTING.md`.
- Always check schema consistency using `npx tsx check_db.ts` when introducing new database fields.
